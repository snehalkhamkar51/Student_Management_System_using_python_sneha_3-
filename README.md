# Student Management System

A simple **Student Management System** built in **Python** using **Linked List** as the data structure.  
This project demonstrates CRUD operations (Create, Read, Update, Delete) and sorting of student records.

---

## Features

- Add multiple students with **roll number, name, and marks**
- Display all students in a table format
- Search student by roll number
- Update student details
- Delete student by roll number
- Sort students by marks (**ascending/descending**)
- Robust **input validation** using exception handling
- Modern Python 3.10+ features with **match-case**
